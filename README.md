## Installation
1. App:
  1. Go to `cd ./cms`
  2. Install all packages `npm install`

3. Server:
  1. Go to `cd ./cms-server`
  2. Install all packages `npm install`

## Run
1. Developer version:
  1. Go to `cd ./cms`
  2. Run file `npm run serve`

2. Distribution version:
  1. Go to `cd ./cms`
  2. Run file `npm run build`

3. Run server:
  1. Go to `cd ./cms-server`
  2. Run by `nodemon`
